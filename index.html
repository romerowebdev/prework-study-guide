Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • We can create bullet points by using "ul" commands.
  • We can list items using "li" commands.
  • We can divide the website using different sections.
  • The only section shown in the front-end is the body.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can change the font size using the styling "font-size".
  • Box-shadow creates a shadow in the box.
  • In CSS we can align the text or boxes wherever we want with the command "align:".
  • With "border:" we can change the thickness, form and color of a border.
  • We can change the "background-color:" to a paragraph, section or the entire website.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds file changes(-A means ALL files)
  • git commit -m "message here": commits new changes to main file
  • git pull origin main: pulls source code from GitHub
  • git pull push origin feature/(feature-name): Pushes changes to GitHub
